Projeto DATASSP CONFIG

Historia e Desenvolvimento

O projeto DATASSP CONFIG foi desenvolvido para complementar o projeto DATASSP, tendo como objetivo facilitar a gestão de conteúdos no site, como metodologias e notas de atualização, de forma dinâmica, sem a necessidade de modificar o código-fonte.

O projeto foi concebido como um site integrado a uma API REST, responsável por armazenar as alterações realizadas e disponibilizá-las posteriormente no site principal.
