Projeto Pizzaria-Veneza

Historia e Desenvolvimento

O Projeto Veneza foi um sistema desenvolvido como trabalho final da disciplina PDSI2 (Projeto e Desenvolvimento de Sistemas de Informação). O objetivo era criar um aplicativo para auxiliar no gerenciamento e controle dos pedidos de uma pizzaria.

O projeto foi desenvolvido principalmente com Flutter, contando com uma API REST criada em Node.js para o armazenamento e gerenciamento dos dados do sistema.
